WORKFLOW EXAMPLES
=================

Example 1: Quick Discovery
1. Click "SADP Tool" button
2. Leave subnet blank (auto-detect) or enter specific subnet
3. Click "Start Scan"
4. View results in Discovery tab
5. Devices are automatically added to Registry

Example 2: Batch Configuration
1. Perform a scan in Discovery tab
2. Go to Batch Operations tab
3. Click "Select All" to select all discovered devices
4. Click "Open in Browser" to configure each device
5. Or "Import to Excel" to add them to your inventory

Example 3: Monitor Network Health
1. Go to Monitor tab
2. Set check interval (e.g., 5 seconds)
3. Click "Start Monitoring"
4. Monitor shows real-time online/offline status
5. Offline counter tracks device availability
6. View trends and device reliability

Example 4: Historical Analysis
1. Devices are tracked in Registry (persistent)
2. Can see "First Seen" and "Last Seen" timestamps
3. Export to CSV for trend analysis
4. Add notes to devices for documentation
