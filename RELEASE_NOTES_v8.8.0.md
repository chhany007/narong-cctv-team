## ğŸ”‘ License Activation

### Obtaining a License Key
Contact the administrator to receive a valid license key for your machine.

### Activating Your License
1. Launch NARONG CCTV v8.8.0
2. When prompted, click "Activate License"
3. Your Machine ID will be displayed
4. Send your Machine ID to the administrator
5. Administrator will generate a license key for your machine
6. Enter the license key in the activation dialog
7. Click "Activate"

### License Types Available
- **PROFESSIONAL (PRO):** Full features, commercial use
- **ENTERPRISE (ENT):** Full features + priority support
- **TRIAL (TRI):** 30-day evaluation, full features
- **DEMO (DEM):** Limited features for demonstration

**Note:** License keys are machine-specific and must be generated by the administrator.
